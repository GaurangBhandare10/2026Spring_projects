# NYC Subway Equity Analysis: Data Dictionary & Outputs

This repository contains the processed datasets and visual outputs used to analyze the relationship between transit service quality and neighborhood demographics in New York City.

## 📁 Data Outputs & Descriptions

Below is a breakdown of each output file generated by the analysis pipeline, its meaning, and its role in testing our core hypotheses.

### 📍 Station & Neighborhood Mapping
| File | Meaning | Project Use |
| :--- | :--- | :--- |
| `station_tract_crosswalk.csv` | A mapping file linking each subway station to its specific Census tract. | Essential join key for combining MTA subway data with US Census demographic data. |
| `tract_demographics.csv` | Contains Census tract-level data including median income, racial composition, and "minority-majority" status. | Used to classify neighborhoods into income quartiles and demographic categories. |

### 🚆 Service & Reliability Metrics
| File | Meaning | Project Use |
| :--- | :--- | :--- |
| `station_peak_service_frequency.csv` | The count of scheduled subway trips during peak rush hours for every station. | Primary variable for **H1** and general service equity comparisons. |
| `line_delay_summary.csv` | Aggregated delay incidents and "delay burden" calculated per subway line over time. | Primary variable for **H2** (Line-level reliability) and **H4** (Shuttle case study). |
| `station_ridership_summary.csv` | Average daily ridership counts segmented by station, year, and day type (weekday vs. weekend). | Used to control for ridership in **H1** and to analyze usage patterns in **H3**. |

### 📊 Integrated Analysis Tables
| File | Meaning | Project Use |
| :--- | :--- | :--- |
| `station_analysis_table.csv` | A master table merging service frequency, ridership, and tract demographics. | The "Source of Truth" for all core station-level statistical modeling. |
| `line_analysis_table.csv` | Combines line-level delay data with ridership-weighted neighborhood income scores. | Used specifically for testing **H2** (Income vs. Delay rate). |

### 📉 Results & Interpretations
| File | Meaning | Project Use |
| :--- | :--- | :--- |
| `hypothesis_results.csv` | The final statistical output, including coefficients, p-values, and significance markers. | Populates the formal "Findings Table" in the final report. |
| `results/figures/*.png` | Data visualizations (Scatter plots, Box plots, and Time-series charts). | Used in reports and presentations to make findings intuitive for non-technical readers. |
