# Report Outline

## 1. Introduction
- State the equity question clearly.
- Explain why a single-fare public transit system can still produce unequal outcomes.

## 2. Data
- Describe each raw dataset.
- Explain the station-to-tract spatial join.
- Explain why GTFS is needed for service frequency.

## 3. Methods
- Define all terms up front.
- State that hypotheses are associative rather than causal.
- Describe each statistical test briefly and justify it.

## 4. Results
- H1 results with coefficient, partial correlation, and plot
- H2 results with Spearman rho and plot
- H3 results with Mann-Whitney statistic, p-value, and confidence interval
- H4 shuttle comparison with charts and caveats

## 5. Discussion
- Summarize whether findings are consistent with inequitable service outcomes.
- Discuss alternative explanations and measurement limits.
- Avoid causal language beyond what the design supports.

## 6. Conclusion
- Restate the main pattern.
- Explain the public-policy relevance.
- Note possible extensions using station accessibility, transfer options, or bus alternatives.
